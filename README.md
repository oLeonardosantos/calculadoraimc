# Calculadora IMC
Com uma interface simples e intuitiva, o app permite que você calcule rapidamente seu Índice de Massa Corporal e descubra em qual faixa de peso você se encontra: abaixo do peso, peso ideal, sobrepeso ou obesidade.

Funcionalidades:
Cálculo rápido e preciso do IMC
Classificação de acordo com a Organização Mundial da Saúde (OMS)
Interface limpa e fácil de usar
Histórico de cálculos (opcional)
Suporte para sistema métrico (kg/m)

Ideal para quem está de olho na saúde, praticando exercícios ou apenas quer se manter informado sobre o próprio corpo.
